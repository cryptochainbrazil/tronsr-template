# Community Plan

We have been working hard to improve TRON's ecosystem, prioritizing the development of dApps for network users and reinvesting part of our rewards into growing the technical team and community.

---

## Contributions to TRON

We are active contributors to the TRON ecosystem since the very beginning:

- Review and suggest improvements to TRON core code
- Active support in development and community channels
- Contributed to TRON's first documentation efforts (Crowdin)
- Active participation in periodic TronCore Meetings
- Helped multiple SRs with node setup during Testnet phase and Mainnet launch
- TronZ MPC participation
- Promoted TRON Meetups across Brazil
- Free dApps training at University of Fortaleza

---

## Products Built for the Network

- **Ledger TRON App** — Official TRON application for Ledger hardware wallets
- **TronLedger Desktop Wallet** — Desktop wallet with Ledger integration
- **GoTron** — Backend API library in Go for TRON blockchain
- **LedgerJS (Tron Module)** — JavaScript library for Ledger-TRON communication
- **Tron Python HD Wallet** — Hierarchical deterministic wallet in Python
- **Python CSV AirDrop** — Tool for batch token distribution via CSV

---

## Education & Content

Technical video streams and training sessions to help new developers build on TRON, covering:

- Blockchain fundamentals
- TRON Virtual Machine (TVM)
- Smart Contract development
- gRPC integration
- Android app development for TRON
